# NVIDIA 購物網站示範

這個專案是以 NVIDIA 為範例的企業購物網站展示。專案展示了一個現代化的電子商務平台，包含產品列表、購物車功能和響應式設計。

## 功能特色

- 展示 NVIDIA 顯示卡等產品的商品目錄
- 適用於桌面和移動設備的響應式設計
- 購物車功能
- 用戶認證
- 產品搜尋和篩選

## 使用技術

- HTML5
- CSS3
- JavaScript
- Bootstrap 響應式設計框架

## 開始使用

1. 複製專案儲存庫
2. 在瀏覽器中打開 `index.html`
3. 開始瀏覽網站

## 專案結構

```
├── index.html          # 主要入口頁面
├── css/               # CSS 樣式表
├── js/                # JavaScript 檔案
├── images/            # 圖片資源
└── README.md          # 專案文件
```
